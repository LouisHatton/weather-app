## Run Locally

This project uses [WetherApi](https://weatherapi.com) to collect information from locations entered by users

- You need to initialise a local variable:

```
REACT_APP_WEATHER_API_KEY="xxxxxxxxxxxxx"
```

To run locally:

- `npm start`
